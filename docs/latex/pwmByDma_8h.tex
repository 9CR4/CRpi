\subsection{Detailed Description}
Generates pwm outputs on the gpio pins using the dma engine. 

To generate a pwm output you first have to call \hyperlink{pwmByDma_8h_af1e4e19586f90d36bad4c295ea042ae6}{pwm\+Dma\+\_\+init()}.~\newline
Then you can initilize a channel with \hyperlink{pwmByDma_8h_adcc4f31a0b62fe334c0a1aad15c19640}{pwm\+Dma\+\_\+init\+Channel()}.~\newline
After this you can add pulses to the channel using \hyperlink{pwmByDma_8h_a98fd2ef532e5e966a7e9e3ee69b022bd}{pwm\+Dma\+\_\+add\+Pulse()}, \hyperlink{pwmByDma_8h_a14578e3f6bcdf9db871ef7530941fa97}{pwm\+Dma\+\_\+add\+Multiple\+Pulse()} or \hyperlink{pwmByDma_8h_a623218011d12638edff77fab8939f7ba}{pwm\+Dma\+\_\+set\+M\+S\+Ratio()}.~\newline
You can clear the output on a port using \hyperlink{pwmByDma_8h_abacb90246d75155b15090e446183a70f}{pwm\+Dma\+\_\+clear\+Port()}~\newline
When you finish you H\+A\+VE TO call \hyperlink{pwmByDma_8h_a43ce520a1f7cff79498d171fbbe83500}{pwm\+Dma\+\_\+free\+Channel()} on the channel you initialized.~\newline
 See \hyperlink{pwmByDma_8c}{pwm\+By\+Dma.\+c} for more details.
