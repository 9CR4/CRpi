\subsection{Detailed Description}
Interface to the clock manager peripheral. 

This peripheral controls various clocks in the B\+C\+M283x So\+Cs, for each of these clocks you can set a clock source and a frequency divider.~\newline
See the C\+L\+K\+\_\+\+S\+R\+C\+\_\+$\ast$ macros for the available clock sources and their speed.
